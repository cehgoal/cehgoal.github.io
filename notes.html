5 HTML Tags I have learned

  • Quote Tag

    • This tag is used to add quotations embedded within other content.

  • Emphasis Tag

    • This tag is used to define empasised text & the content is shown in italic.

  • Abbreviation Tag

    • This tag is used to define the full form of abbreviation or accronym.

  • Image Tag

    • The image tag along with other tags like src(source of image) & alt(alternate text) is used to display image on webpage.

  • Mark Tag

    • This tag is used to highlight important text with yellow colour.
